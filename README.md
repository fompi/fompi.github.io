# fompi.github.io — tarjeta tipo Linktree

## Despliegue en GitHub Pages

1. Crea el repo **fompi/fompi.github.io** (público).
2. Sube estos archivos y carpetas a la raíz del repo.
3. Haz commit y push a `main`.
4. GitHub Pages compilará Jekyll automáticamente.
5. Abre https://fompi.github.io

## Personalización
- Cambia nombre, bio, colores y avatar en `_config.yml`.
- Edita enlaces y orden en `_data/links.yml`.
- Ajusta estilos en `assets/styles.css`.

## Notas
- No usa plugins ni dependencias externas.
- 100% compatible con el entorno por defecto de GitHub Pages.
